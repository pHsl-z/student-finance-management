# 学生财务管理系统

这是一个功能完整的学生财务管理系统，支持收入和支出记录的管理、统计分析和数据可视化。

## 🚀 功能特性

### 💰 财务管理
- **收入记录管理**：添加、编辑、删除学生缴费记录
- **支出记录管理**：管理各类支出项目，支持图标按钮操作
- **收支对比分析**：可视化展示收入与支出趋势

### 📊 数据统计
- **财务概览**：实时显示总收入、总支出、结余
- **分类统计**：按类别统计收支情况
- **趋势分析**：月度收支对比图表

### 🎨 界面优化
- **响应式设计**：适配各种设备屏幕
- **图标按钮**：支出记录操作使用美观的SVG图标
- **删除确认**：所有删除操作都有确认对话框保护

## 🛠 技术栈
- **前端**：HTML5, CSS3, JavaScript (ES6+)
- **UI框架**：Bootstrap 5
- **图表库**：Chart.js
- **数据库**：Supabase (PostgreSQL)
- **构建工具**：Babel (实时转译)
